
## Enrolled Users

The journal users interface lists all users who have one or more roles
on your journal. From this page we can:

- Edit a user

- Add new users

- Add multiple users to roles

- View a user's assignment history  
  - Editorial assignments
  - Review assignments
  - Copyediting assignments
  - Production assignments

![The enrolled users interface. From here we can search, add new users and assign additional roles amongst other things.](../../nstatic/enrolled-users.png)
