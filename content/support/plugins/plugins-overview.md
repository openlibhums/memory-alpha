# Overview of Plugins Available on Janeway

## VAC (Voluntary Author Contributions)
This plugin allows for tracking of VACs (Voluntary Author Contributions). It, however, does **not** enforce APCs (Article Processing Charges). It will at no point block publication of an article, instead it allows editors to track submissions and their linked VACs and VAC waivers. It will also track whether a VAC has been paid.

VACs can be assigned to specific journal sections/article types. When manuscripts are submitted to these sections the author is able to request a VAC waiver if required. This plugin does not affect the publishing workflow in any way.

## Hypothes.is (annotation)
This plugin supports the [Hypothes.is](https://web.hypothes.is/) annotator. It enables users to annotate articles. The Hypothes.is plugin will show in the top-right corner of the web browser from where it can be opened / collapsed.

## Back content import
This plugin imports previously published content into Janeway without going through the full workflow. This is similar to 'Quick submit' on OJS.

## BePress migration
This plugin imports content from [BePress] (https://bepress.com/) into Janeway. It is used for migrations between platforms.

## Books / Multimedia
This plugin enables the publication eBooks as well as other multimedia files such as music on Janeway. It includes access metrics and content can be organised in collections.

## CLOCKSS
This plugin sends content to [CLOCKSS](https://clockss.org/) for archiving purposes.

## Commissions
This plugin enables journals to commission articles from authors. If there is an existing agreement between the editor(s) and author(s), the editor can create the submission (and metadata) and send an invitation to the author. The author can then upload the manuscript.

## Consortial billing
This plugin helps you manage supporter records for a diamond OA funding model. You can set currencies, billing agents, and institution sizes, calculate bands using GNI per capita, and create and edit supporter records and contacts.

## Custom CSS  styling
This plugin enables customisable CSS for the press website. You can add stylesheets for single journals, add them at press level or combine both. It also works with repositories.

## DataCite
This plugin enables minting DOIs through [DataCite](https://datacite.org/) and sending metadata to DataCite.

## DOAJ
This plugin sends metadata to the [Directory of Open Access Journals DOAJ](https://doaj.org/) and can load metadata through the API.

## ePrints migration
This plugin imports content from [ePrints](https://www.eprints.org/uk/) into Janeway. It is used for migrations between platforms.

## Exports (Janeway integrator)
This plugin is used if Janeway is only used for part of the workflow. Most commonly, when Janeway is used up until the prepublication stage at which the articles are exported to another platform for publication.

## Imports
This plugin can be used to import articles, metadata and editorial teams into Janeway. It can also export all of these as .csv files. Additionally, this can be used to batch edit metadata by exporting it, editing the .csv and reuploading it.

## Ingenta migration
This plugin imports content from [Ingenta](https://www.ingenta.com/) into Janeway. It is used for migrations between platforms.

## Preprints
This plugin integrates the Janeway repository platform and journal platform. It allows Janeway to create a preprint in the repository from a manuscript file on the journal platform. Multiple versions of the preprint can be created and peer reviews can be included if needed. All of the aforementioned can be linked to the published manuscript.

## Liquid XSLT
This plugin enables the editing of XSLT on Janeway and to see the effects on articles. Without this plugin, you can still upload XSLT.

## Library of Congress
This plugin sends metadata to the [Library of Congress](https://www.loc.gov/) for archiving purposes.

## Pandoc
This plugin enables the use of [Pandoc](https://pandoc.org/) for 'typesetting'. Manuscript files can be converted into HTML or PDF using Pandoc in the place of using a typesetter.

## Portico
This plugin sends content to [Portico](https://www.portico.org/) for archiving purposes.

## Production transporter (FTP)
This plugin is used to export articles to FTP servers. You can set it up with information on what FTP endpoint to use to then transfer files. This is commonly used to archive through FTP servers.

## Reporting
This plugin enables statistics and metrics to be calculated and displayed within Janeway.

## Scripts
This plugin is part of the developer tools. It supports production scripts to fix issues and makes them reusable.

## Typesetting
This plugin enables the typesetting-stage within Janeway. This is plugin is recommended to be installed by default from version 1.5 onwards.

## Archive
This plugin enables Janeway to be used as an archive. It allows for the multiple versions of an article to be present, as well as functionality for versioning or new editions. This is currently used for an online encyclopedia which periodically releases new versions.

## OJS migration
This plugin imports content from [Open Journal Systems](https://openjournalsystems.com/)(OJS) into Janeway. It is used for migrations between platforms.

## Disqus
This plugin enables the Disqus discussion platform for journal. It is used to add a comment section feature.
