# Clearing the cache

Janeway uses a cache to speed up page loading, you can reset the cache from the Manager page by clicking the **Clear cache** button.

You can clear the cache on a website to ensure you are seeing the most up-to-date version of its content and functionality. Web browsers and web applications store cached data—such as page layouts and images to help pages load more quickly and reduce server load. However, when changes are made to the website; like changes to site settings, themes, plugins, or published content, the cached version may temporarily display outdated information. Clearing the cache removes this stored data and forces the system to retrieve the latest version from the server, ensuring that recent updates appear correctly.
